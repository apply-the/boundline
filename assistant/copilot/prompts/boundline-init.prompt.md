# Command: /boundline-init

Shared guidance: `assistant/README.md`

Map to `boundline init`. For chat-only use, collect `canon_mode_selection`, assistant surfaces, and model routes first, then provide `boundline init --canon-mode-selection <manual|auto-confirm|auto> --assistant <assistant> --route <slot>=<runtime>:<model>`. Preserve workspace-local config output and use CLI config commands for changes.
