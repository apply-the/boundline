# Boundline Guidance Catalog — Phase 3

This package extends the Boundline Guidance Catalog with framework and application-stack guidance.

Included:

- React guidance
- Node.js service framework guidance
- Python service framework guidance
- JVM service framework guidance
- .NET service framework guidance
- Framework guardian rule seeds
- Phase 3 manifest

This package is designed for S2.1 Guidance And Guardian Capabilities.

## Relationship To Previous Phases

Phase 1 introduced cross-cutting guidance:

- Clean Code
- Architecture
- Testing
- Rust
- TypeScript/Node.js

Phase 2 introduced additional language guidance:

- Go
- Python
- JVM/Java
- .NET/C#

Phase 3 adds framework-specific operational guidance.
