# Boundline Guidance Catalog — Phase 2

This package extends the Phase 1 guidance catalog with additional language guidance.

Included:

- Go guidance
- Python guidance
- JVM / Java guidance
- .NET / C# guidance
- Guardian rule seeds for these language families
- A phase manifest describing authority, lifecycle usage, and intended guardian hooks

This package is designed to be consumed by S2.1 Guidance And Guardian Capabilities.

## Relationship To S2.1

S2.1 defines runtime behavior:

- load guidance
- resolve precedence
- execute guardians
- emit structured findings
- expose trace and inspect surfaces

This package provides content:

- language-specific guidance
- idiom rules
- anti-patterns
- guardian seed definitions

## Canon Awareness

These files may be used as:

- Boundline built-ins
- shared expert pack content
- workspace overrides
- Canon-governed standards after promotion

Canon-governed versions are more authoritative, but Boundline must still work without Canon by using pack, workspace, or built-in sources.
